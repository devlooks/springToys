회원

- 회원은 가입하고 회원조회 할 수 있다. 
-  회원은 일반과 VIP 두가지 등급이 있다.
- 회원데이터는 자체 DB 구축, 외부 시스템과 연동 할 수 있다.(추 후, 현재 안됨)

MAIN Todo와 SUB Todo

기본
- 회원은 Todo를 등록 할 수 있다.
- 회원은 Todo를 조회 할 수 있다.
- 조회시, 등록된 순으로 조회 된다.
- 회원은 Todo를 삭제 할 수 있다.
- Todo를 상태를 변경 할 수 있다.(완료, 미완료)

정책 변경
- 등급에 따라 Todo에 Sub Todo를 넣을 수 있다, 
- Sub Todo에도 각각 완료, 미완료, 상태를 가진다.
